Arrastrar y soltar Ficheros
Publicado por gilberto  ·  Última actualización: 15/11/2006 07:44
Autor:gilberto
Fuente: http://www.mygnet.com/articulos/csharp/898/index.php
Bueno, este articulo muestra como permitir arrastrar y soltar un fichero a nuestro Form y mostrar la direccion en un textBox ademas sacamos su icono y lo mostramos en un pictureBox.

 private void Form1_DragEnter(object sender, DragEventArgs e)
        {
            if(e.Data.GetDataPresent(DataFormats.FileDrop))
                e.Effect= DragDropEffects.Copy;
        }

        private void Form1_DragDrop(object sender, DragEventArgs e)
        {
            if (e.Data.GetDataPresent(DataFormats.FileDrop))
            {
                string str = ((string[])e.Data.GetData(DataFormats.FileDrop, true))[0];
                textBox1.Text= str;
                    Icon icono;
                    icono = System.Drawing.Icon.ExtractAssociatedIcon(textBox1.Text);                   
                    pictureBox1.Image = icono.ToBitmap();
                    FileInfo info = new FileInfo(textBox1.Text);
                    lblFileName.Text=info.Name;
            }
        }
